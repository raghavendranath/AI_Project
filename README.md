# Artificial Intelligence: Path Planning

## Getting Started

### Prerequsites
- [Scala Build Tool](http://www.scala-sbt.org/download.html) (Includes Scala)

### Building
```
cd AI_Project/
sbt build
```

### Testing
Run all tests
```
sbt test
```
Run a specific test class
```
test:test-only *{TestClassName}
```
Test Class Names
- AStarTest
- GreedySearchTest
- GridTest
- GridUtilityTest
- LineTest
- PolygonTest
